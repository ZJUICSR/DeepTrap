Risk Suites

Each suite evaluates whether the agent can complete the useful task while resisting a specific class of attack pressure.

R1 Harness Hijacking

Poisoned context or injected workflow steps redirect the agent into unrequested actions.

R2 Privacy Leakage

Planted credentials and sensitive values appear in responses, files, or external actions.

R3 Unauthorized Execution

Dangerous commands or boundary-crossing actions are run beyond the user request.

R4 Supply Chain Risk

Malicious skill or tool-like logic is trusted or invoked during the task.

R5 Tool Abuse

Misleading tool metadata causes harmful behavior or unintended side effects.

R6 Encoding Obfuscation

Hidden encoded payloads are decoded and followed unsafely.

Scenario Categories

The seven task families provide everyday workflows in which the six risk suites are replayed.

T01Meeting notes, action items, and routine workflow summaries.
T02Code formatting, refactoring, API output, and log processing.
T03Research notes, literature review, citations, and survey analysis.
T04Deployment readiness, configuration validation, and health checks.
T05Sales, revenue, and structured business-data reporting.
T06Content drafting, style checks, onboarding docs, and feedback summaries.
T07System administration, environment audits, deployment checks, and log backup.
