Red-Teaming Agent Execution Contexts

Open-World Security Evaluation on OpenClaw

Abstract

Agentic language-model systems increasingly rely on mutable execution contexts: files, memory, tools, skills, and auxiliary artifacts. DeepTrap evaluates whether such contexts can induce unsafe behavior while preserving benign task completion in OpenClaw agents.

The public release contains a 42-case benchmark spanning six vulnerability classes and seven operational scenarios, with attack and utility grading scores for comparing target models.

Benchmark

DeepTrap packages the public runner, replay tasks, and scoring code needed to evaluate OpenClaw models. The private task-generation and attack-search pipeline is not included in this repository.

Framework

Vulnerability Classes

R1 Harness Hijacking

Poisoned context or injected workflow steps redirect the agent into unrequested actions.

R2 Privacy Leakage

Planted credentials and sensitive values appear in responses, files, or external actions.

R3 Unauthorized Execution

Dangerous commands or boundary-crossing actions are run beyond the user request.

R4 Supply Chain Risk

Malicious skill or tool-like logic is trusted or invoked during the task.

R5 Tool Abuse

Misleading tool metadata causes harmful behavior or unintended side effects.

R6 Encoding Obfuscation

Hidden encoded payloads are decoded and followed unsafely.

Usage

Install the repository and run the bundled benchmark tasks against an OpenClaw-configured model.

git clone https://github.com/ZJUICSR/DeepTrap.git
cd DeepTrap
pip install -e .
python benchmark.py --dry-run
python benchmark.py --model openrouter/anthropic/claude-sonnet-4-5 --suite R1
